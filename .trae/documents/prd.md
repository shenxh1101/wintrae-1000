## 1. 产品概述

消防管理 Web 应用，面向园区安全员和物业主管，提供隐患排查、巡检管理、演练组织、档案维护等全流程数字化管理能力，降低火灾风险、提升应急响应效率。

- 解决传统纸质台账效率低、隐患跟踪易遗漏、巡检路线不规范等痛点
- 目标用户：园区安全员（日常执行）、物业主管（审批监督）

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 安全员 | 管理员分配账号 | 登记隐患、执行巡检、参与演练、维护档案 |
| 物业主管 | 管理员分配账号 | 审批分派、查看总览、导出台账、监督整改 |

### 2.2 功能模块

1. **总览页**：风险等级仪表盘、待办数量统计、近期报警列表、快捷入口
2. **隐患页**：拍照登记、分派整改、期限提醒、复查关闭
3. **巡检页**：按楼栋生成路线、记录灭火器/喷淋/通道状态、巡检报告
4. **演练页**：安排计划、签到、评分、问题汇总
5. **档案页**：建筑管理、消防设施、责任人、维保记录、证书到期提醒、月度台账导出

## 3. 核心流程

### 隐患处理流程

安全员在日常巡检或发现隐患时，通过拍照登记创建隐患记录。物业主管收到通知后进行审批分派，指定整改责任人和期限。整改责任人完成整改后提交复查。安全员进行现场复查，确认整改合格则关闭隐患，不合格则打回重新整改。超期未整改的隐患自动触发提醒。

```mermaid
flowchart TD
    "发现隐患" --> "拍照登记"
    "拍照登记" --> "主管审批分派"
    "主管审批分派" --> "整改执行"
    "整改执行" --> "提交复查"
    "提交复查" --> "复查是否通过?"
    "复查是否通过?" -->|"通过"| "关闭隐患"
    "复查是否通过?" -->|"不通过"| "整改执行"
    "整改执行" -->|"超期"| "期限提醒"
    "期限提醒" --> "整改执行"
```

### 巡检流程

选择楼栋后系统自动生成巡检路线，安全员按路线逐项检查灭火器、喷淋、通道状态，记录异常项并拍照。巡检完成后系统自动生成巡检报告。

```mermaid
flowchart TD
    "选择楼栋" --> "生成巡检路线"
    "生成巡检路线" --> "逐项检查"
    "逐项检查" --> "记录状态与照片"
    "记录状态与照片" --> "是否还有检查点?"
    "是否还有检查点?" -->|"是"| "逐项检查"
    "是否还有检查点?" -->|"否"| "生成巡检报告"
```

### 演练流程

物业主管安排演练计划，配置评分项。演练当日参与人员签到，演练过程中逐项评分并记录问题。演练结束后自动生成演练总结报告。

```mermaid
flowchart TD
    "安排演练计划" --> "配置评分项"
    "配置评分项" --> "演练当日签到"
    "演练当日签到" --> "执行演练"
    "执行演练" --> "逐项评分"
    "逐项评分" --> "记录问题"
    "记录问题" --> "生成演练总结"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深红 (#DC2626) 象征消防，搭配暖灰 (#1F2937) 营造专业感
- **辅助色**：琥珀色 (#F59E0B) 用于警告，翠绿 (#10B981) 用于正常/完成状态
- **按钮风格**：圆角矩形（8px），主按钮实色填充，次按钮描边
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- **布局风格**：左侧固定导航栏 + 右侧内容区，卡片式模块布局
- **图标风格**：线性图标（Lucide Icons），2px 描边，与文字等宽

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 总览页 | 风险等级仪表盘 | 环形进度图，中心显示等级文字，颜色从绿到红渐变 |
| 总览页 | 待办统计卡片 | 四列网格卡片，每张含图标+数字+标签，悬停微上浮 |
| 总览页 | 近期报警列表 | 垂直时间线，左侧时间、右侧事件卡片含状态标签 |
| 总览页 | 快捷入口 | 四个图标按钮横排，圆角卡片样式 |
| 隐患页 | 隐患列表 | 数据表格，行悬停高亮，状态列彩色标签 |
| 隐患页 | 拍照登记弹窗 | 居中模态框，照片上传区虚线边框，表单字段垂直排列 |
| 隐患页 | 期限提醒 | 顶部固定提示条，渐变背景色 |
| 巡检页 | 楼栋选择 | 左侧竖向标签页，选中项左侧红色指示条 |
| 巡检页 | 巡检路线 | 步骤条组件，已完成/当前/待检查三种状态 |
| 巡检页 | 检查记录 | 卡片式检查项，三列网格，状态图标+文字 |
| 演练页 | 演练计划列表 | 卡片网格，每张含状态标签+时间+地点 |
| 演练页 | 签到管理 | 进度环+人员列表，实时更新到场比例 |
| 档案页 | 建筑档案 | 卡片列表，缩略图+基本信息 |
| 档案页 | 证书到期提醒 | 表格行颜色区分：正常白底、临期黄底、过期红底 |
| 档案页 | 月度台账导出 | 月份选择器+导出按钮，点击生成CSV下载 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1280px
- 导航栏在窄屏下收起为图标模式
- 卡片网格在中等屏幕下从四列调整为两列
- 表格在窄屏下支持横向滚动

### 4.4 动效设计

- 页面切换：淡入淡出（200ms）
- 卡片悬停：微上浮 + 阴影加深
- 数据加载：骨架屏占位
- 状态变更：颜色渐变过渡
- 数字变化：计数动画
